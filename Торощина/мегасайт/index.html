Файлы с расширением .html открываются браузером.

Все файлы html состоят из тегов.

Справочник тегов.

  • Теги для текста
  • Теги для выделения текста
  • Теги для графических элементов
  • Теги, задающие структуру страницы

1.Теги для текста

  • p - абзац, от англ. paragraph
  • h1..h6 - заголовки, от англ. heading
  • ul/ol - списки, маркированный и нумерованный, от англ. unordered ordered list, li - пункты списка, от англ. List Item
  • a - якорь от англ. anchor Подробнее

2.Теги для выделения текста

  • b - жирный, от англ. bold
  • i - курсивный, от англ. italic
  • del - перечеркнутый, от англ. delete

3.Теги для графических элементов

  • hr - линия, от англ. horizontal line
  • img - картинка, от англ. image

4.Теги, задающие структуру страницы

  • !DOCTYPE html - не тег, а инструкция для браузера
  • html - заключает в себя все содержимое веб-страницы
  • head - для хранения служебной информации о странице
  • body - содержимое html документа
  • header - раздел шапки сайта
  • main - весь основной контент
  • section - раздел веб-страницы
  • footer - раздел подвала сайта
  • nav - раздел с меню
  • div - универсальный блок
